About Arc Agent

Arc Agent is a native iOS and iPadOS application that provides on-the-go access to AI-powered sprint intelligence reports. The app analyzes GitHub pull requests across multiple repositories and generates comprehensive sprint reports using advanced AI technology.

Getting Started

1. Installation

Download Arc Agent from TestFlight using the invitation link provided by your organization.

2. Configuration

On first launch, you'll need to configure:

  • API Endpoint: Your organization's Sprint Intelligence API URL
  • API Key: Authentication key provided by your administrator
  • GitHub Token: Personal access token with repository read permissions

3. Running Analysis

  1. Tap "Analyze Sprint" on the main screen
  2. Configure the date range and repositories (or use defaults)
  3. Watch real-time progress as the analysis runs
  4. View the formatted report when complete

Troubleshooting

Cannot Connect to API

Solution:

  • Verify your API endpoint URL is correct (should start with https://)
  • Check that your API key is valid and hasn't expired
  • Ensure you have an active internet connection
  • Contact your administrator to verify API access

GitHub Authentication Failed

Solution:

  • Verify your GitHub personal access token is valid
  • Ensure the token has repo or public_repo permissions
  • Check that the token hasn't expired
  • Generate a new token at github.com/settings/tokens

Analysis Takes Too Long

Solution:

  • Large repositories or date ranges may take several minutes
  • The app will continue processing in the background
  • You can close the app and check back later
  • Consider reducing the date range or number of repositories

Report Not Displaying

Solution:

  • Ensure the analysis completed successfully
  • Check the progress screen for any error messages
  • Try refreshing the report list
  • Verify your API endpoint is accessible

App Crashes or Freezes

Solution:

  • Force quit the app and relaunch
  • Ensure you're running the latest version from TestFlight
  • Restart your device if problems persist
  • Clear cached data in Settings if available

Privacy & Security

Data Storage

Arc Agent stores the following data locally on your device:

  • API Credentials: Stored securely in iOS Keychain
  • Cached Reports: Stored in app sandbox for offline access
  • User Preferences: Settings and configuration stored locally

Data Transmission

All communication with the Sprint Intelligence API uses HTTPS encryption. Your GitHub token is transmitted securely and never stored on remote servers.

Permissions

Arc Agent requires:

  • Internet Access: To communicate with the Sprint Intelligence API
  • Keychain Access: To securely store API credentials

The app does NOT require or request:

  • Location services
  • Camera or photo library access
  • Contacts or calendar access
  • Microphone access

System Requirements

  • iOS/iPadOS: Version 17.0 or later
  • Devices: iPhone, iPad (Universal app)
  • Storage: 50 MB minimum
  • Internet: Required for analysis, optional for viewing cached reports

Frequently Asked Questions

Is Arc Agent free?

Arc Agent is provided by your organization. Contact your administrator for licensing and access information.

Can I use Arc Agent offline?

You can view previously cached reports offline. Running new analyses requires an internet connection.

How do I get a GitHub token?

Visit github.com/settings/tokens and create a personal access token with repo or public_repo permissions.

Can I analyze private repositories?

Yes, if your GitHub token has repo scope (full repository access) and you have permission to access those repositories.

How long are reports stored?

Reports are cached locally on your device until you clear app data or uninstall the app. Server-side retention is managed by your organization.

Does Arc Agent work on Apple Watch or Mac?

Currently, Arc Agent is available for iPhone and iPad only. Apple Watch and macOS versions are planned for future releases.
